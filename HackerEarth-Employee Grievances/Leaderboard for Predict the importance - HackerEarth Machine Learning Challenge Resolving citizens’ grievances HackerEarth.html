Ranking Criteria

Developers are ranked according to the total score. Developers who have the same total score are ranked by least effective time.

The effective time is the sum of the time consumed for each problem solved. The time consumed for a solved problem is the time elapsed from the beginning of the challenge to the submission of the first accepted solution. If there is a penalty time in minutes, this duration is also added for every rejected run for that problem regardless of submission time. There is no time added for a problem that is not partially/correctly solved.
